QU'EST-CE QUE LE G-SCORE ? Le G-Score mesure l'instabilité thermodynamique d'une réponse IA. Pas la vérité — la cohérence interne. 0 = stable · 1 = instable Trois signaux mesurés : · Cohérence — la logique interne tient-elle ? · Dérive — le texte reste-t-il sur le sujet ? · Stabilité — les affirmations se contredisent-elles ? Chaque réponse reçoit un score. Vous définissez le seuil. L'API observe ou agit.

CAS D'USAGE — LEGALTECH Un cabinet d'avocats utilise un LLM pour générer des résumés de contrats. Sans gouvernance : les réponses vont directement aux clients — avec un risque de stabilité inconnu. En mode OBS : chaque réponse est scorée avant livraison. G-Score > 0.6 — SIGNAL — transmis à un associé senior, non envoyé au client. En mode GOV : les réponses instables sont automatiquement retenues ou réécrites. Meilleure réponse. Aucun changement dans le workflow client. Résultat : — risque sémantique contenu — responsabilité légale réduite — trace d'audit complète générée — conformité EU AI Act intégrée, pas ajoutée
