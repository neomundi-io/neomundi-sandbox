WHAT IS G-SCORE? The G-Score measures the thermodynamic instability of an AI response in real time. Built on Lyapunov stability theory and Friston's Free Energy Principle - the same mathematics used to model self-regulating biological systems. 0 = stable output · 1 = high risk Every response gets a score. You set the threshold. The API decides. This score did not exist before. Now you can track it, log it, and act on it.
